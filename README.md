# ERPHMS - Sistem Manajemen Pasien Gawat Darurat

Program C++ untuk mengelola antrian pasien di IGD dengan implementasi:
- Priority Queue (antrian berdasarkan prioritas)
- Stack (riwayat kunjungan)
- Linked List (riwayat medis)
- Binary Search Tree (indeks pasien)
- Array (data master dokter & obat)

## Cara menjalankan
1. Compile dengan g++: `g++ ERPHMS.cpp -o erphms.exe`
2. Jalankan: `./erphms.exe` (Linux/Mac) atau `erphms.exe` (Windows)

## Fitur
- Tambah/lihat/layani pasien
- Ubah/hapus pasien
- Manajemen dokter & obat
- Riwayat kunjungan & medis
